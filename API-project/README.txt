API Documentation for the sample webclient that consumes a public API service

This Api was built from the medium's Api documentation accessible via (https://github.com/Medium/medium-api-docs). 

It contains 2 files mypub.js and myStory.js
When mypub.js runs on a node command line, it gets a JSON formated list of the publications that i subscribed to on Medium.

When myStory.js is ran on a node command line, it POSTS a new story to my profile on Medium.
